# Персональный сайт *Романа* *Турусова*
### Выпускной проект курса по веб-разработке от Loftschool

---

Инструкция по установке:
```
git clone https://github.com/RomaTur/ls-sitePortfolio.git
cd ls-sitePortfolio
npm i
```
